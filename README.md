# Holberton — React & TypeScript Exercises

Build and practice React fundamentals using TypeScript. This repository contains focused exercises and small projects designed to teach idiomatic TypeScript in a React environment.

- **Author:** Patrick Macabulos
- **Primary focus:** React + TypeScript exercises and learning material

## 🚀 Project Overview

This repo collects short, opinionated exercises that cover common React patterns with TypeScript: components, props & state typing, hooks, context, forms, testing, and simple app composition. Each exercise is self-contained under the `Typescript/` folder and includes instructions and example solutions where applicable.

## 🧭 Contents

- `Typescript/` — exercises and small projects (core material)

## 🛠 Prerequisites

- Node.js (16+) and npm or Yarn
- Basic familiarity with JavaScript, React, and TypeScript

## ▶️ Quick Start

1. Clone the repo

```bash
git clone <repo-url>
cd holbertonschool-web_react
```

2. Install dependencies (if a given exercise has its own package.json)

```bash
npm install
# or
yarn install
```

3. Open the exercise folder under `Typescript/`, follow the README for that exercise, and run the provided scripts (build, start, test).

## 📁 Repository Structure

- `Typescript/`
	- `task_1/` — exercise description, starter code, tests
	- `task_2/` — another exercise
	- `...`

Each exercise directory should include a short README with goal, constraints, and verification steps.

## ✅ How to Complete an Exercise

1. Read the exercise README and requirements.
2. Implement the TypeScript-typed React code in the `src/` or `index.tsx` provided.
3. Run unit tests (if provided) or `npm start` to run locally.
4. Commit your solution to a feature branch and open a PR if collaborating.

## 🧩 Contributing

- Fork the repo or create branches for your work.
- Keep changes small and focused per exercise.
- Add or update exercise README files with clear acceptance criteria.

## 🧪 Testing

Where present, exercises include basic unit tests. Run `npm test` or check the exercise README for specifics.
